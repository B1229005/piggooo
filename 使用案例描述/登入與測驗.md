## 登入與測驗

| 欄位 | 描述內容 |
| :--- | :--- |
| **使用案例名稱** | **進行旅行人格測驗 (手勢互動)** |
| **行動者** | 使用者、後端伺服器 |
| **前置條件** | 使用者已啟動 App，進入測驗引導畫面。 |
| **正常情節** | 1. 系統顯示測驗題目卡片。<br>2. 使用者閱讀題目後，透過手勢滑動卡片。<br>3. 若使用者**往右或往左滑動**，系統分別記錄為「是」或「否」；若使用者**往上滑動**，系統記錄為「SoSo」。<br>4. 系統切換至下一題，直到所有題目作答完畢。<br>5. 系統打包作答紀錄，發送 API 請求至後端伺服器。<br>6. 伺服器回傳分析結果，畫面跳轉並顯示使用者的「專屬旅行人格」。 |
| **例外情節** | **5a. 網路斷線或伺服器無回應：**<br>系統捕捉網路連線錯誤，將測驗結果暫存於本地端 (Local Storage)，並跳出提示：「網路似乎有點問題，我們已保留您的進度，請稍後重試。」 |
| **後置條件** | 成功產出並記錄使用者的旅行人格，進入系統首頁。 |
