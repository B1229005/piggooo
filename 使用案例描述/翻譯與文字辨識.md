## 翻譯與文字辨識

| 欄位 | 描述內容 |
| :--- | :--- |
| **使用案例名稱** | **拍照擷取相片文字 (OCR) 翻譯** |
| **行動者** | 使用者、手機相機、Google Vision API、翻譯 API |
| **前置條件** | 使用者開啟翻譯面板，且設備支援相機功能。 |
| **正常情節** | 1. 使用者點擊翻譯面板中的「相機」按鈕。<br>2. 系統呼叫 ImagePicker 開啟相機，使用者拍攝外文照片。<br>3. 系統將相片壓縮並轉碼為 Base64 格式。<br>4. 系統呼叫 Google Vision API (TEXT_DETECTION)。<br>5. API 成功回傳辨識出的文字字串，系統將其填入文字框。<br>6. 系統自動觸發 MyMemory API 進行翻譯，顯示於下方區塊。 |
| **例外情節** | **4a. 照片中找不到文字：**<br>Vision API 回傳成功但 annotations 為空，系統將文字框更新為：「照片中找不到文字喔！」。<br>**4b. API 連線錯誤：**<br>系統捕捉 Exception，擷取錯誤代碼並顯示：「辨識失敗，請檢查網路或 API 設定...」。 |
| **後置條件** | 使用者獲得照片文字的翻譯結果與相片預覽。 |
