## 天氣與行程探索

| 欄位 | 描述內容 |
| :--- | :--- |
| **使用案例名稱** | **搜尋其他城市地區天氣** |
| **行動者** | 使用者、Geocoding API、Open-Meteo API |
| **前置條件** | 使用者點擊天氣卡片，進入天氣詳細預報面板。 |
| **正常情節** | 1. 使用者在搜尋列輸入欲查詢的城市名稱（如：東京）。<br>2. 系統顯示載入中動畫。<br>3. 系統呼叫 Geocoding API 將城市名稱轉換為經緯度座標。<br>4. 系統將座標傳入方法，向 Open-Meteo 請求該區天氣。<br>5. API 回傳結果，系統解析目前氣溫、濕度、降雨機率及 24 小時預報。<br>6. 系統隱藏載入動畫，並更新畫面上的天氣圖示與數據。 |
| **例外情節** | **3a. 查無此城市或 API 錯誤：**<br>Geocoding API 回傳空陣列，系統捕獲錯誤，結束搜尋狀態並清空搜尋列，維持原本的天氣畫面不變。<br>**1a. 返回當前定位：**<br>使用者點擊「定位圖示」，系統呼叫 Geolocator 獲取真實 GPS 並重抓當地天氣。 |
| **後置條件** | 天氣面板更新為使用者指定城市的精確氣象資訊。 |
